## 17. Technical Infrastructure

The backend is built on Node.js with Express.js, using Redis as a key-value store for data management. The platform is hosted on Render with auto-deploy from GitHub. All blockchain operations run on XRPL with XUMM/Xaman SDK for wallet integration.

The backend provides RESTful APIs for user authentication, meme submission and tracking, staking operations, MNFT minting and marketplace, battle arena operations, DAO voting, and leaderboards.

The frontend is built with HTML, CSS, and JavaScript, hosted on WordPress and Hostinger. Wallet connection uses XUMM WebSocket, with state management handled through LocalStorage and API calls.
